Simple Guide to Hosting Projects on Github.

Step-by-step easiest guide for hosting a website on Github pages.

This article will cover how to host your HTML, CSS & Javascript file on Github pages. If you want to create a great portfolio, then I would suggest creating your profile using HTML, CSS & Javascript.

You should only use frameworks like REACT, Angular or VUE when you have to create huge applications with many many components and pages etc.

Step — 1: Creating a Project

First things first you will have to create your project, from scratch or whatever way you like. Only one thing to keep in mind keep the main HTML file’s name as index.html as seen in the picture below.

Step — 2: Create a Repository on Github

Firstly go and create a new repository.

Give your project a name and description, make sure to include a readme. Most of the time if you see some readme in any project you are more likely to trust the project.

Step-3: Upload the files on Github

Upload files on Github.

Wait for them to finish uploading….

Step-4: Settings [Final Step]

Go to the settings tab in Github.

Scroll down and find the Github Pages option in settings. And select a main branch or any other branch that you would like to publish and host online.

Then click save…

Then, just wait until GitHub does all the hard work…. Grab a cup of coffee or anything that you would snack on, after like 5 minutes refresh the github.

When you see the above thing rejoice and pat yourself on the back. Your site is published share the link to anyone, put it on Linkedin etc.. etc.

Enjoy …
